Welcome to the Sulfur Framework modding docs!

Sulfur Framework is a Roblox Sonic framework with a modding API that allows users to mod in custom levels, objects, and behaviour in general via Lua and our own bindings.
NOTE: The modding API uses Lua, not Luau. So any new syntax is *NOT* supported, such as +=, -=, and for loops without pairs()
